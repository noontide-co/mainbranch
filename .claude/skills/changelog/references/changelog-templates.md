# Changelog Templates

Templates for CHANGELOG.md files by business type.

---

## Base Template (All Types)

```markdown
# Business Changelog

Track record of **[Business Name]** metrics and milestones.

**Cadence:** [Weekly/Monthly/Quarterly]
**Business Type:** [Type]
**Started:** [YYYY-MM-DD]

---

## Metrics Tracked

| Metric | Description |
|--------|-------------|
| [Metric 1] | [What it measures] |
| [Metric 2] | [What it measures] |
| [Metric 3] | [What it measures] |

---

## [Year]

### [Month Year]

**Date:** [YYYY-MM-DD]

| Metric | Value | Change |
|--------|-------|--------|
| [Metric 1] | [value] | [+/- or --] |
| [Metric 2] | [value] | [+/- or --] |
| [Metric 3] | [value] | [+/- or --] |

**Notes:**
- [Optional context about this period]

---
```

## Entry Update Template

When adding a new entry, append this structure:

```markdown
### [Month Year]

**Date:** [YYYY-MM-DD]

| Metric | Value | Change |
|--------|-------|--------|
| [Metric 1] | [value] | [+X / -X / --] |
| [Metric 2] | [value] | [+X / -X / --] |
| [Metric 3] | [value] | [+X / -X / --] |

**Notes:**
- [Context about this period]

---
```

### Change Formatting

- First entry: Use `--` for change column
- Subsequent entries: Show absolute change with + or -
- Example: `+$500`, `-3`, `+15%`

### Year Headers

When a new year starts, add a new year header:

```markdown
---

## 2027

### January 2027

...
```
