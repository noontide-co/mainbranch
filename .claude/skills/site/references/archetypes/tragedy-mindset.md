# Tragedy Mindset

Nothing can change, fate is sealed. Belief, not circumstance. Use to describe the worldview the offer breaks.

## When to invoke

- As `audience_current_archetype`, never as `archetype` for the offer.
- To name the belief that's keeping them stuck (different from circumstance).

## Do-not-state list

- "Don't be so negative." (Pep-talk; reads as dismissal.)
- "Mindset is everything." (Cliche; ages fast.)

## Paired-imagery template

- The closed-loop image (calendar full of the same week, the same desk in the same light) paired with one frame where something is different — small but specific.

## Reframe target

- Pair with `archetype: david-goliath` (the giant is vulnerable; here's how) or `archetype: wounded-healer` (the belief is breakable; I broke it).
