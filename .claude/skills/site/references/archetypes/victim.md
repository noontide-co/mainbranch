# Victim

Pure suffering, no agency yet. The audience may currently feel like a Victim — but **never position the buyer as a Victim on a sales page.** This archetype's role is to describe the trap the offer breaks, not the buyer.

## When to invoke

- As `audience_current_archetype` (what they're trapped in), never as `archetype` for the offer.
- To describe the systemic problem the offer reframes.

## Do-not-state list

- "You're a victim." (Even if true, the buyer rejects this label on a sales page.)
- "It's not your fault." (Will be heard as condescension.)
- "The system is broken." (Without specifics, this is empty.)

## Paired-imagery template

- The trap made tangible (the system / pattern / scenario keeping them stuck) paired with absence of the operator.

## Reframe target

- Pair with `archetype: savior` (we'll do it for you) or `archetype: wounded-healer` (I was where you are).
