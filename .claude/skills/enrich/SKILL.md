# Enrich (Deprecated)

This skill has been merged into `/think`.

---

## What to Use Instead

The `/think` skill now handles all enrichment workflows through its **codify mode**.

### Examples

| Old Command | New Command |
|-------------|-------------|
| `/enrich` | `/think add context to my files` |
| `/enrich testimonials` | `/think add new testimonials` |
| "Update my reference files" | `/think codify` or `/think enrich` |

---

## Why the Change

One skill for the full knowledge workflow:

```
Research -> Decide -> Codify (includes enrich)
```

This reduces cognitive load - you don't need to remember which skill does what.

---

## Quick Start

Just run:

```
/think
```

And say what you want to add or update. The skill routes to codify mode automatically when you mention:
- "add context"
- "enrich"
- "I have new info"
- "update reference files"
- "add testimonials"

See `/think` for full documentation.
