# Organic Frameworks

Structures for Reels, TikTok, carousels. Organic = value-first, engagement over conversion.

---

## Core: Hook-Retain-Reward

```
HOOK (0-3s): Stop scroll. Curiosity or recognition.
RETAIN (3-45s): Deliver value. Maintain attention.
REWARD (last 5-15s): Payoff + soft CTA.
```

Algorithms favor watch time and saves.

---

## Framework 1: Educational

**Structure:** Hook → Tips → Takeaway

**Best for:** How-to, lists, tutorials

| Section | Purpose |
|---------|---------|
| Hook | Promise value ("3 things I wish I knew...") |
| Setup | Brief context (optional) |
| Tips 1-3 | One idea per beat |
| Takeaway | Summarize |
| CTA | Soft ask |

**Timing (30-60s):**
```
Hook: 3s | Setup: 5s | Tips: 8s each | Takeaway: 5s | CTA: 3s
```

---

## Framework 2: Story-Based

**Structure:** Hook → Trigger → Journey → Outcome → Result

**Best for:** Personal narratives, lessons learned

| Section | Purpose |
|---------|---------|
| Hook | Tease transformation |
| Trigger | What started it |
| Journey | The struggle |
| Outcome | What you discovered |
| Result | Where you are now |
| CTA | Invite engagement |

**Timing (45-90s):**
```
Hook: 3s | Trigger: 10s | Journey: 20-40s | Outcome: 10s | Result: 5s | CTA: 3s
```

---

## Framework 3: Transformation

**Structure:** Before → Turning Point → After

**Best for:** Journey content, before/after

| Section | Purpose |
|---------|---------|
| Hook | Tease the gap |
| Before | Where you started |
| Turning Point | What changed |
| After | Where you are now |
| Bridge | How they can too |
| CTA | Soft ask |

**Timing (30-60s):**
```
Hook: 3s | Before: 10s | Turning: 10s | After: 10s | Bridge: 10s | CTA: 3s
```

---

## Framework 4: Problem-Solution

**Structure:** Hook (Problem) → Agitate → Solution → Proof

**Best for:** Pain points, myth-busting, "stop doing X"

| Section | Purpose |
|---------|---------|
| Hook | Name the problem |
| Problem | Describe the pain |
| Agitate | Make it worse |
| Solution | The fix |
| Proof | Why it works |
| CTA | Engagement ask |

**Timing (30-45s):**
```
Hook: 3s | Problem: 8s | Agitate: 8s | Solution: 10s | Proof: 8s | CTA: 3s
```

---

## Framework 5: Contrarian

**Structure:** Bold Claim → Evidence → Reframe → New Perspective

**Best for:** Challenging wisdom, sparking discussion

| Section | Purpose |
|---------|---------|
| Hook | Bold claim |
| Evidence | Support it |
| Reframe | Shift perspective |
| New Perspective | What to do instead |
| CTA | Invite debate |

---

## Choosing Framework

| Content Type | Framework |
|--------------|-----------|
| How-to, tutorial | Educational |
| Personal story | Story-Based |
| Before/after | Transformation |
| Myth-busting | Problem-Solution |
| Bold opinion | Contrarian |

## Hook Formulas

**Curiosity:**
- "The one thing [successful people] do that you don't"
- "This changed everything for me"
- "Here's why [common advice] is wrong"

**Recognition:**
- "POV: You just realized [relatable thing]"
- "You're not [negative label]. You're just [reframe]"

**Controversy:**
- "Unpopular opinion: [bold claim]"
- "Stop [common behavior]. Here's why."

**Story:**
- "The day I [pivotal moment]"
- "I almost [gave up], then this happened"

---

## Retention Techniques

**Pattern Interrupts (every 5-8s):**
- Change camera angle
- Add text overlay
- Shift energy/tone
- Jump cuts

**Open Loops:**
- "But that's not the crazy part..."
- "Wait, it gets better..."

**Value Stacking:**
- Each beat = its own payoff
- Give 80% value before asking

---

## Soft CTAs

| Hard (Avoid) | Soft (Use) |
|--------------|------------|
| Link in bio | Save for later |
| Sign up now | Follow for more |
| Limited time | Comment [word] for more |
| DM to work together | Tag friend who needs this |

---

## Platform Notes

**Instagram Reels:**
- 15-30s optimal (up to 60 for story)
- First frame = grid preview
- Saves > likes

**TikTok:**
- 60-90s OK with retention
- Hook works with sound off
- Stitch/duet = more reach

**Both:**
- Watch time > views
- Saves > likes
- Reply to comments fast
