# Proof

Testimonials, case studies, and evidence that your offer delivers.

---

## How to Use

Ad skills look for proof in this folder. Organize testimonials so Claude can find the right proof for each angle.

---

## Testimonial Format

For each testimonial, capture:

```markdown
## [Customer Name]

**Context:** [Who they are, what situation they were in]
**Result:** [Specific outcome they achieved]
**Quote:** "[Their exact words]"
**Date:** [When collected]
**Angle fit:** [Which angles this supports]
**Media:** [Screenshot, video link, or text]
```

## How to Collect

1. **Ask after wins** — When someone shares a result, ask to quote them
2. **Screenshot DMs** — With permission, save social proof
3. **Exit surveys** — "What would you tell someone considering this?"
4. **Video testimonials** — Record short clips when possible
5. **Specific > vague** — "Lost 20 lbs" beats "Great program"

---

## Proof Quality Checklist

- [ ] Specific outcome (numbers, timeframes)
- [ ] Named customer (or credible anonymous)
- [ ] Addresses a real objection
- [ ] Matches an angle we run
- [ ] Permission to use
