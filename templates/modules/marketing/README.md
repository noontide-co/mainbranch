# Marketing Module

For those running ads, creating content, or marketing a Skool community.

## When to Use

Copy this module if you:
- Run Facebook/Meta ads
- Create content (video scripts, copy)
- Sell a course, community, or membership

## Structure

```
marketing/
├── offer.md          → What you sell, price, benefits
├── audience.md       → Who buys, pains, desires
├── proof/            → Testimonials, case studies
└── angles/           → Messaging entry points
```

## Why This Structure

Skills like `ad-static`, `ad-video-scripts`, and `skool-vsl-scripts` look for these files. Consistent structure = skills just work.

## Files

### offer.md
What you're selling. Price points. What's included. CTA URL.

### audience.md
Who buys. Their pains. Their desires. What they've tried. What they believe.

### proof/
Testimonials with names and specifics. Case studies. Screenshots. Results.

### angles/
Different entry points to the same offer. Each angle = different pain/desire emphasis.
