# Marketing Module

For those running ads, creating content, or marketing a Skool community.

## When to Use

Copy this module if you:
- Run Facebook/Meta ads
- Create content (video scripts, copy)
- Sell a course, community, or membership

## Structure

```
marketing/
├── offer.md           → What you sell, price, benefits
├── audience.md        → Who buys, pains, desires
├── psychographic.md   → Deep buyer psychology
├── language-bank.md   → Verbatim customer phrases
├── campaign-tracker.md → Performance log + learnings
├── proof/             → Testimonials, case studies
│   └── README.md
└── angles/            → Messaging entry points
    ├── README.md      → Angle index + avatar matrix
    └── angle-template.md
```

## Why This Structure

Skills like `ad-static`, `ad-video-scripts`, and `skool-vsl-scripts` look for these files. Consistent structure = skills just work.

## Files

### offer.md
What you're selling. Price points. What's included. CTA URL.

### audience.md
Who buys. Their pains. Their desires. What they've tried. What they believe.

### psychographic.md
Deep buyer psychology. Emotional state. Beliefs. Who does NOT fit.

### language-bank.md
Real phrases from customers organized by intent. Source of voice authenticity.

### campaign-tracker.md
Log of active campaigns, winners, losers, and learnings. The feedback loop.

### proof/
Testimonials with names and specifics. Case studies. Screenshots. Results.

### angles/
Different entry points to the same offer. Each angle = different pain/desire emphasis. Includes avatar matrix for targeting.

## Setup Order

1. Fill `offer.md` — What are you selling?
2. Fill `audience.md` — Who's buying?
3. Fill `psychographic.md` — Go deeper on buyer psychology
4. Start `language-bank.md` — Collect 10-20 customer quotes
5. Create first 3 angles in `angles/` — Map to avatars
6. Add proof to `proof/` — At least one testimonial per angle
7. Launch first campaign — Log in `campaign-tracker.md`
8. Iterate — Update based on learnings
