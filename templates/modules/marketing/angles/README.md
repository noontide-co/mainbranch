# Ad Angles

Proven messaging angles for your ads. Each angle targets a specific customer pain or desire.

---

## How to Use

1. Pick an angle that matches your campaign goal
2. Create ads using that angle's hooks, proof, and copy blocks
3. Log performance in `campaign-tracker.md`
4. Iterate on winners, learn from losers

## Angle File Template

Each angle should have its own file with:
- 5-10 hooks
- 2-3 copy blocks (short, medium, long)
- Relevant proof points
- Best practices for this angle

Use [angle-template.md](angle-template.md) to create new angles.

---

## How to Build Angles

1. **Start with customer language** — What pains/desires repeat?
2. **Group by theme** — Overwhelm, trust, speed, results, community
3. **Name each angle** — Make it memorable and clear
4. **Assign proof** — Each angle needs at least one testimonial
5. **Map to avatars** — Not all angles work for all segments
6. **Test and iterate** — Log winners/losers in campaign tracker
