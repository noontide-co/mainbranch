# Testimonials

Customer proof with consent status.

---

## Usage Rules

Before using a testimonial in ads:

1. **Consent:** Must have documented consent for ad use
2. **Typicality:** If outcome-based, need typicality disclosure (see `compliance/typicality/`)
3. **Match:** Testimonial must match offer tier being sold
4. **Currency:** Should be from within last 2 years
