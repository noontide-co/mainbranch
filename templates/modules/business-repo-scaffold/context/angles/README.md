# Angles

Messaging entry points for campaigns.

---

## What Is An Angle?

An angle is a specific "door" into your offer. Same product, different framing based on what resonates with different segments or mindsets.

Example angles for a credit repair community:
- **Resolution failure** — "91% of resolutions fail by February"
- **Fresh start** — "January is the perfect time to begin"
- **System vs willpower** — "Why systems beat motivation"

---

## Angle File Format

Each angle should be its own file: `[angle-slug].md`

Contents:
```markdown
# [Angle Name]

## Core Insight
One sentence that captures the angle's "aha moment."

## Hook Themes
- [Hook idea 1]
- [Hook idea 2]
- [Hook idea 3]

## Key Messages
- [Message 1]
- [Message 2]

## Proof Points
Which testimonials/data support this angle?

## Audience Segment
Who does this angle resonate with most?
```

## Creating New Angles

Sources for angle inspiration:
1. Customer language (what phrases do they use?)
2. Competitor ads (what angles are being tested?)
3. Seasonal/timely hooks (what's happening now?)
4. Pain point variations (different aspects of the problem)
5. Benefit variations (different outcomes they want)
